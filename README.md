# POP Historical Crawler

抓取[財政部國有財產署](http://www.fnp.gov.tw)各分區之標租不動產歷年開標結果資訊。
_（目前只在網站上找到北區歷年資料)_

- [北區分署](http://www.fnpn.gov.tw/ct/CFT.php?page=CFTMain2&area=N000)

## Required

- Install nodejs
- Install npm

## Quick Start

#### Install
```sh
$ npm install
```

#### Start
```sh
node ./pop_historical_crawler.js
```
